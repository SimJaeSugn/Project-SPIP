# 백로그
- 프로젝트의 경로 복사 기능 추가
- VScode 열기 할때 PATH 에 없다고 안되는거 같다. PATH 에 추가하는 방법을 같이 알려주거나, 설치된 경로를 직접 추가할 수 있도록 설정에 추가(향후 다른 툴도 추가할 수 있도록 확장성을 고려한다. )
- 카드들 드레그해 위치 이동 가능하도록 한다.
- 카드를 즐겨찾기에 추가할 수 도록한다.
- 트레이바에 추가 ( 우클릭시 대시보드,종료,즐겨찾기 기능 제공 ) 
    - 대시보드 : 프로그램이  실행되면 뜨는 대시보드 창 (대시보드는 닫아도 트레이는 남아 있어야 한다. )
    - 종료 : SPIP 프로그램 전체 정료 (창, 트레이)
    - 즐겨찾기 : 즐겨찾기에 등록된 카드를 좌우로 슬라이딩 하며 볼 수 있도록 우측하단에 UI제공

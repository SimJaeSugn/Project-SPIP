# 브리핑 AI 연동

## 모델 정보
base_url = http://127.0.0.1:1234/v1/chat/completions
모델명 : exaone-3.5-7.8b-instruct

## 기능 요구사항 
- 오늘의 브리핑을 AI를 이용해 피드백 받도록 수정 ( 피드백은 스트리밍 방식으로 출력한다. )
    - Langchain 을 이용하고 openai 4mini 모델로 최초 개발 ( 키는 .env 참조 )
    - 관리되는 모든데이터를 대상으로 변경이 일어날 경우. 피드백이 필요하다 판단되는 경우 피드백을 생성
    - 온프레미스 모델을 연결 할 수 있도록 확장을 고려하여 개발 ( openai api 호환 인터페이스 )
    - 단순히 처리해야할 일을 정리하는 수준을 넘어 처리하는 방법을 가이드하는 정보도 포함

## goal
- 사용자가 알아야 할 내용, 알면 좋은 내용, 급하게 처리하지 않으면 안될 내용 등을 이벤트 발생과 동시에 브리핑한다.
- 새로운 이벤트가 발생하면 거의 즉각적으로 반응해야한다.
- 새로운 이벤트가 발생하면 앞서 생성한 피드백의 내용 중 처리가 되지 않은 내용을 포함한 피드백이 생성되어야 한다.

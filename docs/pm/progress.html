데스크톱 전환 (Electron) — 완료 · 검증 통과

로컬 웹앱(브라우저 접속) → Electron 설치형 데스크톱 앱으로 이전 완료. 사용자가 매번 터미널·브라우저를 거치던 불편을 없애는 것이 전환 이유다. 전송 계층을 HTTP → Electron IPC로 바꾸고, 도메인 로직은 (args,ctx)→result 순수 함수로 분리해 헤드리스 단위테스트한다. 설계 정본: electron-migration.html v1.1(§4 IPC 계약·§6 보안).

리스크 / 유의사항

VS Code 열기(R-12) — 코드·분기·테스트는 완료이나 이 PC에 code CLI 미설치로 실제 창 오픈은 라이브 미실측. code가 PATH에 있는 환경에서 동작.
대시보드 시각 회귀 — 브라우저 자동화 도구 부재로 픽셀 단위 시각 검증은 정적·라이브 헤더·순수함수 시뮬레이션까지만. 실제 브라우저 스모크 권장.
저위험·정보성 — 감사 로그·레이트리밋 부재(L-OBS-1, 단일 사용자 루프백이라 실위험 낮음, 멀티유저 확장 시 도입 권고).
